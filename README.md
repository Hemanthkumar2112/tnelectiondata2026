# Tamil Nadu Assembly Election 2026 — Awareness Dashboard

![Dashboard Preview](https://img.shields.io/badge/Status-Complete-brightgreen)
![Version](https://img.shields.io/badge/Version-1.0.0-blue)
![Data Source](https://img.shields.io/badge/Data-ECI%202026-maroon)

A comprehensive, interactive, single-file static dashboard for the **Tamil Nadu Legislative Assembly (TNLA) 2026** election. This tool is designed to empower voters with real-time data, candidate profiles, and electoral insights.

## 🚀 Key Features

*   **Interactive District Map:** Spatial visualization of voter density across Tamil Nadu with top 15 district rankings highlighted.
*   **Candidate Explorer:** Advanced filtering by Name, District, Constituency, Party, Gender, Criminal Records, and Education.
*   **Deep-Dive Candidate Profiles:** Detailed views of candidate affidavits including Education, Occupation, Assets, Liabilities, and Criminal Case details.
*   **Nomination Funnel:** Constituency-wise breakdown of nominations (Filed vs. Accepted vs. Contesting).
*   **Electoral Insights:** Age distribution, party statistics, and voter demographics.
*   **Awareness Hub:** Key dates, voting procedures, and "Do's and Don'ts" for the 2026 election.

## 📊 Dataset

The dashboard is powered by a massive dataset (~20GB) hosted on Hugging Face, containing structured candidate data, photos, and affidavit summaries.

**Hugging Face Dataset:** [TN Election 2026 Data Hub](https://huggingface.co/datasets/Hemanth-thunder/tnelection2026data)

## 🛠️ Technology Stack

*   **Frontend:** HTML5, Vanilla JavaScript (ES6+), CSS3 (Modern Flex/Grid).
*   **Visualizations:** [Chart.js](https://www.chart.js.org/) for interactive charts.
*   **Mapping:** Custom SVG/D3-inspired mapping engine for Tamil Nadu districts.
*   **Icons:** FontAwesome 6.
